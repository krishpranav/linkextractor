# linkextractor
this tool will extract all links in a website

Steps to run this tool:

      1. git clone https://github.com/krishpranav/linkextractor
      2. cd linkextractor
      3. sudo chmod +x *
      4. bash install.sh
      5. python3 linkextractor.py
      
TESTED ON MAC AND LINK 100% working
